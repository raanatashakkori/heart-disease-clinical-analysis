# Heart Disease Clinical Analysis
**Predictive Modeling for Cardiovascular Risk Assessment using Clinical Data**

## Overview
A comprehensive machine learning pipeline to analyze and predict heart disease
based on clinical features. Aimed at supporting Clinical Informatics and
healthcare decision-making.

## Dataset
[Heart Disease Dataset — Kaggle](https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset)  
Place `heart.csv` in the project root before running.

## Technologies
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## How to Run
# 1. Clone repo
git clone git@github.com:raanatashakkori/heart-disease-clinical-analysis.git
cd heart-disease-clinical-analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run
python heart_disease_analysis.py
