# Heart Disease Clinical Analysis
*** Predictive Modeling for Cardiovascular Risk Assessment using Clinical Data ***
## Overview
This project develops a comprehensive machine learning pipeline to analyze and predict heart disease based on clinical features. Aimed at supporting **Clinical Informatics** and healthcare decision-making.
## Dataset
Heart Disease Dataset

## Technologies
Python, Pandas, Scikit-learn, matplotlib, SHAP
